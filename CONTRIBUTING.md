# Contributing to Open Source Europe

Thank you for contributing to the OSE community.

## Code of Conduct

All contributors are expected to follow our [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to abide by its terms. To report a violation, email [lukasz@opensourceeurope.org](mailto:lukasz@opensourceeurope.org).

## AI Usage Policy

All contributors must follow our [AI Policy](AI-POLICY.md). The short version:

- If you used AI tools, disclose it in the PR description using `Generated-by: <Agent Name and Version>`
- Review all AI-generated content before submitting — you are fully responsible for it
- Never add an AI agent as a `Co-Authored-By:` commit author — agents are tools, not authors
- Governance votes, CoC enforcement, and security decisions are human-only

## Commit Messages

Use [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/):

```
<type>: <short description>

[optional body]
```

Types: `feat`, `fix`, `docs`, `ci`, `chore`, `refactor`

## Pull Requests

1. Fork the repository and create a feature branch from `main`
2. Make focused, well-described commits
3. Open a pull request against `main`
4. Fill in the PR template completely, including the AI disclosure section

## Reporting Issues

Use the issue templates. Include enough context for maintainers to understand and act on the report.
